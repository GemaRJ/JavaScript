Binding

Par poder traer elementos que estan en lo lógica// app.ts y representarlo en la parte gráfica se utiliza one-way binding.Por ejemplo vamos a renderizar una variable: {{ nombre }} y voy a sacar un {{ nota }}

El event binding hace lo contrario. Lanza un evento y pasa los datos desde lo gráfico a lo lógico. Para ello en el button, select, input asociamos un evento (click, change, input aso) ademas de definir el metodo en la lógica (app.ts).
